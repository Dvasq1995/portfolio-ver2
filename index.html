Local Library

Link to repo: Local Library

Link to live app: Local Library

This is an application that displays several stats of a library.

It displays popular books, authors, and most common genres.

The user can select the links in each stat to see the list of books in that category.

To build this application, I created several functions that used higher order functions within their body blocks with JavaScript that handled different arrays of objects.

Said functions also used other helper functions as well as functions that wrote to the DOM to display return values to the viewport.

This application is for anyone who would like to know simple stats given they're storing data in a compatible format.

From building this application, I learned the value of higher order functions as well as helper functions which ultimately shorten up code while maintaining readibility.

Technology Stack used:

  1. HTML
  2. CSS
  3. JavaScript

Decoder Ring

Link to repo: Decoder Ring

Link to live app: Decoder Ring

This application allows a user to encode or decode messages using three different ciphers.

The user can select either the Caesar Shift, the Polybious Square, or the substitution cipher to decode or encode a message.

To build this application, I created JavaScript functions and helper functions that defined storage of different alphabet grids for encoding and decoding purposes as well as character codes using the UTF-16 Table as reference.

I also used test driven development with Chai and Mocha to make sure my functions did exactly what I wanted them to do.

This code worked with JavaScript code that helped output content onto the DOM.

This application is for anyone who would like to send encoded messages and decode encoded messages that used these ciphers

Basically, anyone who'd like to send secret messages to someone.

From building this application, I learned that using tests against your functions really helps to prevent getting unexpected outputs from them. I also became more appreciative of the NYC npm package that how much code my tests are covering.

Technology Stack used:

  1. HTML
  2. CSS
  3. JavaScript
  4. Mocha
  5. Chai
  6. NYC
